Introduction

Welcome to the CubeXell CubeSolver documentation.

This project provides a programmable Rubik’s Cube solver designed for speed, flexibility, and educational use.

The documentation is organized like a handbook. Each page covers a specific concept, allowing developers to quickly find solutions and understand how the system works.

What is CubeSolver?

CubeSolver is a software system that:

  • Represents Rubik’s Cube states
  • Applies legal cube moves
  • Finds solutions to scrambled cubes
  • Provides a programmable solving engine

Example

Cube cube = new Cube();

cube.scramble();

Solver solver = new Solver();

String solution = solver.solve(cube);

System.out.println(solution);
