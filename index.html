Overview

The Cubexell CubeSolver is a powerful and efficient algorithm for solving Rubik's cubes. It provides an easy-to-use interface for both beginner and advanced users. The solver supports various cube configurations and implements state-of-the-art algorithms for optimal performance.

For more information, visit the official GitHub repository: Cubexell CubeSolver Repository.

Installation

To get started with Cubexell CubeSolver, you need to install the necessary dependencies. Follow the instructions below:

Prerequisites

  • Python 3.6+
  • pip (Python's package installer)

Steps

  1. Clone the repository: 
    git clone https://github.com/randomcuber/cubesolverCampFull.git
  2. Navigate to the project directory: 
    cd cubesolverCampFull
  3. Install dependencies: 
    pip install -r requirements.txt

Usage

Once the CubeSolver is installed, you can begin using it to solve cubes. Here’s a simple example of how to use the library:

Basic Example


import cubesolver

# Define a scrambled cube state (in a valid format)
scrambled_cube = "UDLRFBL..."

# Solve the cube
solution = cubesolver.solve(scrambled_cube)

# Display the solution
print(solution)

This will return a list of moves required to solve the cube.

Advanced Usage

For more advanced functionality, you can customize the solver with different algorithms, optimization options, and more.


# Advanced solver options
advanced_solution = cubesolver.solve(scrambled_cube, method="Fridrich", optimize=True)
print(advanced_solution)
