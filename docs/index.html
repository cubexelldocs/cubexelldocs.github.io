Introduction

Welcome to the CubeXell CubeSolver documentation. This documentation is designed to be clear, fast to navigate, and easy to extend as the project grows.

What is CubeSolver?

CubeSolver is a programmable Rubik's Cube solving engine that can:

  • Represent cube states
  • Apply legal cube moves
  • Scramble cubes
  • Compute solving sequences

Example

Cube cube = new Cube();

cube.scramble();

Solver solver = new Solver();

String solution = solver.solve(cube);

System.out.println(solution);
