Introduction

CubeSolver is a programmable Rubik's Cube solving engine designed for speed, clarity, and extensibility.

Installation

# Clone repository

git clone https://github.com/randomcuber/cubesolverCampFull

cd cubesolverCampFull

Quick Start

Cube cube = new Cube();

cube.scramble();

Solver solver = new Solver();

String solution = solver.solve(cube);

Cube State

The cube state represents the positions and orientations of all pieces.

Moves

Moves follow standard cube notation such as R, U, F, L, D, and B.

Solver Algorithm

The solver computes sequences of legal moves that transform a scrambled cube into the solved state.

Basic Usage

Cube cube = new Cube();

cube.apply("R U R'");

Advanced Usage

Advanced usage includes custom scrambles, performance tuning, and debugging solver states.
